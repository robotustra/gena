Эта программа это прошивка для пушпульного генератора с 3 каналами. Первый канал
два канала имеют скважность менее 50% и период 50 герц. Они сдвинуты по фазе на 180 градусов.
Третий канал работает на частоте 100 герц и этот синхроимпульс может смещаться относительно фронтов основных имульсов

----------------
Новая задача:

Сделать основной генератор на 1000 Гц,
Синхронный с ним генератор 50 герц, и еще один канал синхронизируюший импульс на 50 герц, синхронный со вторым генератором.
